# Cognitive Architectures for Language Agents (语言智能体的认知架构)

> Survey date: 2026-03-05.

---

**Core idea:** Design LLM agents using principles from cognitive science — modular memory (episodic, semantic, working, procedural), structured action spaces, and generalized decision-making processes. Hybrid neuro-symbolic approaches analogous to Kahneman's "fast" and "slow" thinking.

**Key frameworks:**
- **CoALA (Cognitive Architectures for Language Agents)** — foundational framework combining LLMs with symbolic AI memory and decision-making
- **EGI Framework** — five-layer cognitive architecture: Perception → Semantic Axes → Agents → Orchestration → Meta-Cognition
- **CCA (Cognitive Control Architecture)** — lifecycle supervision framework for aligned AI agents

**Key proponents:** Academic community (multiple arXiv papers 2025-2026); ICLR 2026 Workshop "MemAgents" on memory for agentic systems.

**Critical distinction:** Agent memory ≠ LLM memorization. Agent memory is online, interaction-driven, and under the agent's control. Requires write policies during interaction, temporal credit assignment, and consolidation into generalizable knowledge.

**Why it matters:** As agents become more autonomous and long-lived, principled memory substrates supporting single-shot learning, context-aware retrieval, and knowledge consolidation become essential.
