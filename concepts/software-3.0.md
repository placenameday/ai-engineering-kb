# Software 3.0 (软件3.0)

> Survey date: 2026-03-05.

---

**Core idea:** A new software paradigm where natural language is the programming language. Three eras:
- **1.0:** Traditional code (explicit instructions in Python/C++)
- **2.0:** Neural networks (train models with data, coined by Karpathy 2017)
- **3.0:** LLM-powered (prompts are programs, English is the programming language)

**Key proponent:** Andrej Karpathy (Y Combinator AI Startup School keynote, June 2025).

**Key concepts within:**
- **LLMs as operating systems** — analogous to 1960s mainframes: centralized, expensive, time-shared
- **The Autonomy Slider** — user adjusts AI control from minimal assistance to full autonomy
- **LLM Psychology** — LLMs as "people spirits" with "jagged intelligence," encyclopedic knowledge but cognitive deficits (hallucinations, anterograde amnesia)
- **Generator-Verifier Loop** — the core engineering challenge: build the verification as good as the generation

**Why it matters:** Y Combinator reported 25% of Winter 2025 startups had codebases 95% AI-generated. Software creation is being democratized but requires new engineering discipline.
