# Conceptual Framework Design for AI Engineering KB

## Design Decision: Option D (Hybrid)

### Rationale
The 13 concepts span fundamentally different abstraction levels and concern types. A pure tree (current state) forces false hierarchies. A pure lifecycle model loses the "what kind of knowledge is this?" dimension. The hybrid approach uses:

1. **Vertical axis**: 5-layer stack from paradigm vision down to production safety
2. **Horizontal axis**: Cross-cutting concerns that span multiple layers
3. **Reading path**: Numbered sequence for newcomers

### The 5 Layers

| Layer | EN Name | ZH | Question Answered | Concepts |
|-------|---------|-----|-------------------|----------|
| L1 | Paradigm & Vision | 范式与愿景 | Why is everything changing? | Software 3.0, Vibe Coding -> Agentic Engineering |
| L2 | System Architecture | 系统架构 | How do you structure AI systems? | Compound AI Systems, Cognitive Architectures, LLM Orchestration |
| L3 | Design Patterns | 设计模式 | What patterns solve recurring problems? | Agentic Design Patterns, Anthropic Agent Design, Agentic Patterns (Willison) |
| L4 | Engineering Methodology | 工程方法论 | What process and principles guide development? | Eval-Driven Development, Spec-Driven Development, 12-Factor Agents |
| L5 | Safety & Governance | 安全与治理 | How do you deploy and operate responsibly? | Guardrails & Progressive Autonomy |

### Cross-Cutting Concern
Context Engineering spans L2-L5 -- it is the core discipline that affects architecture decisions, pattern selection, methodology, and safety.

### Learning Path
1. Software 3.0 (orient)
2. Vibe Coding -> Agentic Engineering (self-locate on spectrum)
3. Compound AI Systems (architecture mindset)
4. Agentic Design Patterns (pattern vocabulary)
5. Context Engineering (core skill)
6. Anthropic Agent Design (design philosophy)
7. Cognitive Architectures + LLM Orchestration (deepen architecture)
8. 12-Factor Agents (production principles)
9. Spec-Driven Development + Eval-Driven Development (methodology)
10. Agentic Patterns (Willison) (practitioner wisdom)
11. Guardrails & Progressive Autonomy (safety layer)
